# Open Paws Fundraising — Agent Quick Reference

Donor churn prediction (CatBoost) and grant matching (semantic search) platform. Privacy-first: zero PII in the feature table, pseudonymous donor_id throughout.

## How to Run

```bash
# Install
pip install -e ".[dev]"

# Start Streamlit dashboard
streamlit run dashboard/app.py
```
